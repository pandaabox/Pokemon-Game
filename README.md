# Pokemon-Game
A browser-based simple pokemon battle game, cloned from Pokémon FireRed, made using html, css, and javascript.

# Features 
- Turn-based battle game
- Status effects
- Battle messages with typing animation
- Health bars with animation

# How to Play
1. Open 'index.html' in a browser.
2. Select an attack.
3. Defeat your opponent.

# Limitations
- Limited pokemon
- Not responsive yet
